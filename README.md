POST Bookmarker
===========
This Chrome extension lets you create bookmarks with POST payload, also working on mobile devices (but you shouldn't be able to create them).

Chrome web store
-----------
This Chrome Extension is published on the [Chrome web store](https://chrome.google.com/webstore/detail/post-bookmarker/iefcddnfnoleclapjmcihejbfjadopbf).
